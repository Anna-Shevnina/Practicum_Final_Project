# Практический блок диплома. Составление SQL-запроса и автоматизация теста.
## Работа с базой данных
Необходимо проверить отображениение заказа, созданного в базе данных, и убедиться, что статусы заказов корректно записываются в базу данных
Файл с SQL- запросами приложен. 
## Автоматизация теста
Необходимо автоматизировать подготовленный сценарий, который проверяет получение данных о заказе по его треку.
- Для запуска теста должны быть установлены пакеты pytest и requests
- Запуск теста выполняется командой pytest
